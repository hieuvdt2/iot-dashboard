# Ánh xạ mã F (bảng) ↔ tên trên sơ đồ use case

Dùng **bảng này** trong báo cáo Word (dưới Hình 2.1a), **không bắt buộc** ghi F1, F2… trên hình.

| Mã | Tên chức năng (Bảng 2.2) | Trên sơ đồ 2.1a / 2.1b |
|:--:|---------------------------|-------------------------|
| F1 | Giám sát realtime | Giám sát thời gian thực |
| F2 | Lưu lịch sử dữ liệu | Tra cứu lịch sử theo ngày |
| F3 | Cảnh báo môi trường | Cảnh báo ngưỡng môi trường |
| F4 | Đánh giá tình trạng vườn | Đánh giá tình trạng vườn |
| F5 | Điều khiển bơm | Bật / tắt máy bơm |
| F6 | Chế độ AUTO | Chế độ tưới tự động (AUTO) |
| F7 | Cấu hình ngưỡng | Cấu hình ngưỡng cảnh báo |
| F8 | Cấu hình mẫu cây trồng | Chọn mẫu cấu hình theo loại cây |
| F9 | Xác thực người dùng | Đăng nhập / đăng ký |
| F10 | Phân quyền | Quản lý phân quyền |
| F11 | Hỗ trợ đa nền tảng | Sử dụng trên Web, Mobile, Desktop |
| F12 | Đồng bộ cloud | Gửi dữ liệu cảm biến / Ghi Firebase (2.1b) |
